# Restaurant "122" — React + Express + MongoDB

Ville: Sidi Bel Abbès — Cuisine: Shawarma  
Langues: Français (par défaut) + Arabe

## Démarrage
Backend:
```
cd server
cp .env.example .env
npm install
npm run seed
npm run dev
```
Frontend:
```
cd client
cp .env.example .env
npm install
npm run dev
```
